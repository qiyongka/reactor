将电抗器数据分为两种，将哈吐一线，吐哈一线，吐巴二线的数据分析，
分析能否得到与平均值相近的点是是否在同一个平面，或者在同一个区域，或者同一个点。

1.dist.py  :将九组电抗器数据读出来，计算得到相均值，计算每个采样点与相均值之间的距离，将所有的采样点按照距离排序并且输出

2.dist_square.py  :将九组电抗器数据读出来，然后按照能量的思想，将所有的频率分量全部平方，
计算得到相均值，计算每个采样点与均值之间的距离，将所有的采样点按照距离排序并且输出

3.dist_standard.py  :将九组电抗器数据读出来，计算得到相均值，然后根据100Hz的数据进行归一化，将每个采样点以100Hz做最大值做归一化，计算每个采样点与均值之间的距离，将所有的采样点按照距离排序并且输出

4.dist_square_standard.py  :将九组电抗器数据读出来，然后按照能量的思想，将所有的频率分量全部平方，
计算得到相均值，然后根据100Hz的数据进行归一化，将每个采样点以100Hz做最大值做归一化，计算每个采样点与均值之间的距离，将所有的采样点按照距离排序并且输出


距离的计算公式有两种：
1.sigma（abs（array1 - array2））
2.欧氏距离：sqrt（（array1 - array2）*（array1 - array2））
